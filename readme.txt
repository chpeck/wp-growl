== Installation ==

1. Upload `wp-growl.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

** Premium Setup **
1. Get a token for BuzzGrowl Premium from
[BuzzGrowl](http://buzzgrowl.com/)
1. Enter your BuzzGrowl Premium token in the 'BuzzGrowl Configuration'
page under the 'Plugins' menu
